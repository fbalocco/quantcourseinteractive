RDM Quantitative Track

Interactive Modules

by Francesco Balocco

Standalone interactive companions for each session deck. Each module includes an explorable walkthrough, quizzes, a case study, and a webR test bench for simulated data and guided analysis in R.

Session 1B: Intro to Quantitative Research

Probability, distributions, sampling, and confidence intervals.

Session 3: Experiments and Comparing Means

Experimental design, t-tests, ANOVA, and assumptions.

Session 4A: Archival Data and OLS

Correlation, regression intuition, diagnostics, and controls.

Session 5A: Advanced Regression

Nonlinearities, logit/probit, and panel data intuition.

Session 6A: Survey Research

Sampling, reliability, validity, and factor analysis.
Each module is fully self-contained. To use the R test benches you will need internet access to load webR from the official CDN and a modern browser that supports OffscreenCanvas for plotting.
